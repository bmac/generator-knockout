# Generator knockout
A [knockout](http://knockoutjs.com/) generator for [Yeoman](http://yeoman.io).

## Getting started
- Make sure you have [yo](https://github.com/yeoman/yo) installed:
    `npm install -g yo`
- Install the generator: `npm install -g generator-knockout` *(locally works too)*
- Run: `yo knockout`
